# KLIK KLIK — Klikklik.lv jaunais dizains

Custom CSS + JS premium dizainam, kas tiek izmantots klikklik.lv mājaslapā uz Pixieset platformas.

## Faili

- **kknew.css** — visi globālie stili (Manrope fonts, krāsas, sekciju klases, animācijas, mobile nav, Vimeo lightbox)
- **kknew.js** — visa funkcionalitāte (dinamiska navigācija, scroll reveal, swipe galerijas, Vimeo modāļi, kontaktforma)

## Lietošana caur jsDelivr CDN

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/laganovskisuldis/klikklik@main/kknew.css">
<script src="https://cdn.jsdelivr.net/gh/laganovskisuldis/klikklik@main/kknew.js" defer></script>
```
